== README

Conservative estimates of daily rent easily double the standard monthly amount we pay for housing.
This can stand as a major impediment to individuals rehousing themselves
and hopes of obtaining employment.
That's why I present Roof Be Known.
A hub for space sharing to drastically reduce costs,
by avoiding the effects of premiums meant for getaways.


Please feel free to use a different markup language if you do not plan to run
<tt>rake doc:app</tt>.
